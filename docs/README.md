# Math formulas

## Функции

### Circle

#### Функция area(a)
Возвращает площадь круга
Пример вызова:
area(5) вернет 25пи

#### Функция perimeter(r)
Возвращает длину окружности 
Пример вызова:
perimetr(5) вернет 10пи

### Square 

#### Функция area(a)

#### Функция perimeter(r)
