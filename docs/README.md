# Math formulas

## Функции

### Circle

#### Функция area(a)
Возвращает площадь круга
Пример вызова:
area(5) вернет 25пи

#### Функция perimeter(r)
Возвращает длину окружности 
Пример вызова:
perimetr(5) вернет 10пи

### Square 

#### Функция area(a)
Возвращает площадь квадрата
Пример вызова:
area(5) вернет 25

#### Функция perimeter(r)
Возвращает периметр квадрата
Пример вызова:
perimetr(5) вернет 20

### Rectangle

#### Функция area(a)
Возвращает площадь прямоугольника 
Пример вызова:
area(5, 4) вернет 20

#### Функция perimeter(r)
Возвращает периметр прямоугольника
Пример вызова:
perimetr(5, 4) вернет 18
